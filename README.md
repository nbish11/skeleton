Boilerplate code for developing a PHP application/library.

## Install

Via Composer

``` bash
$ composer require nbish11/skeleton
```

## Usage

``` php
$skeleton = new nbish11\Skeleton();
echo $skeleton->echoPhrase('Hello, League!');
```
